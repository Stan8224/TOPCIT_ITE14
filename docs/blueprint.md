# **App Name**: GradeView

## Core Features:

- Data Extraction: Automatic data extraction from the linked GitHub repository (https://github.com/Stan8224/TOPCIT_ITE14.git) to identify student submissions, assignments, and associated data. This data may also include project descriptions, deadlines, test cases, evaluation metrics and grading rubrics
- Review Submission UI: Enable teachers to easily grade the code through the UI, providing easy options to submit marks.
- Personalized Feedback Generation: Generate constructive and specific feedback leveraging an LLM, based on identified patterns and discrepancies, and considering grading rubrics; serves as a tool to expedite and enhance grading consistency. The tool decides when it is appropriate to point to existing grading rubrics or project documentation.
- Data storage: Persisting review and grade information for the project submissions

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) to convey professionalism and focus.
- Background color: Light Gray (#F0F3F4) to ensure readability and a clean interface.
- Accent color: Teal (#3498DB) to highlight key actions and interactive elements.
- Body font: 'Inter', a grotesque-style sans-serif providing clarity and modern appeal for detailed views and summaries.
- Headline font: 'Space Grotesk', a proportional sans-serif to present a futuristic feel to the UI.
- Crisp, minimalist icons representing grades, feedback types, and action items.
- Smooth transitions for navigation and subtle loading animations to keep students engaged.
